Quick tip: Find and replace with Neovim

Use telescope live_grep to search for what you want to replace: Populate the results to the quick fix list with ctrl + q Then run find and replace: :cdo s/search/replace/g The :cdo will execute the specified command in every entry in the quick fix list. After that, every entry have to be saved, you can go in every buffer and save manually or to save every buffer: :cdo update References Quickfix Telescope

GO - Tipos compostos

Array Um array é uma sequencia de tamanho fixo de zero ou mais elements var q [3]int = [3]int{1,2,3} q := [...]int{1,2,3} // com a reticencia o tamano será determinado pela quantidade de inicializadores O tamanho de um array faz parte do seu tipo q := [3]int{1,2,3} q = [4]int{1,2,3,4} // erro de compilação: não é permitido atribuir [4]int a [3]int Arrays são inflexiveis Fatias (slices) Uma fatia representa um sequencia de tamanho variavel cujo elementos tem o mesmo tipo.
